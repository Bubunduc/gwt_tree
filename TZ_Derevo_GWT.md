# Техническое задание

## Цель

Разработать приложение на **Java + GWT** для управления древовидной
структурой узлов.

### Стек

-   Java 8
-   GWT 2.7
-   Ant
-   PostgreSQL
-   MyBatis

## Этап 1. In-Memory

### Модель узла

-   id
-   parentId
-   name
-   ip
-   port

При старте создается корневой узел:

-   id = 0
-   parentId = null
-   name = rootNode
-   ip = 127.0.0.1
-   port = 1111

Удаление корневого узла запрещено.

### Функционал

-   Отображение дерева с бесконечной вложенностью.
-   Отображение свойств выбранного узла.
-   Таблица со всеми узлами.
-   Добавление узла.
-   Добавление дочернего узла.
-   Редактирование `name`, `ip`, `port`.
-   Удаление узла с каскадным удалением всех потомков.
-   После любого изменения дерево и таблица должны обновляться.

### Ограничения

-   Пустые значения не допускаются.
-   Использование готовых реализаций дерева запрещено.


## Этап 2. PostgreSQL + MyBatis

Заменить хранение данных в памяти на PostgreSQL без изменения
интерфейса.

### Таблица

`client_node`

Поля: - id - parent_id - name - ip - port

### Mapper

-   insert(Node)
-   update(Node)
-   delete(id)
-   findById(id)
-   findAll()

## Результат

Приложение должно поддерживать полный CRUD для древовидной структуры и
корректно работать как с In-Memory хранилищем, так и с PostgreSQL через
MyBatis.
